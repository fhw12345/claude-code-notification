# 🔔 cc-plugin-notification — 让你不再错过 Claude Code 的回复

用 Claude Code 跑任务的时候切去看别的东西，然后忘了回来看结果，10 分钟后才想起来？我经常这样。

所以写了个小插件：**Claude 回复完成后，任务栏图标会闪橙色 + 播放提示音**。就这么简单，不弹窗不打扰，就是提醒你该回去看了。

**支持：** Windows Terminal、VS Code 集成终端

**安装（2 条命令）：**
```
claude plugin marketplace add fhw12345/claude-code-notification
claude plugin install cc-plugin-notification@cc-notification-marketplace
```

**可配置：** 通知级别（all / normal / important）、勿扰时段、声音开关。默认配置开箱即用，不需要额外设置。

Repo + 文档：https://github.com/fhw12345/claude-code-notification
